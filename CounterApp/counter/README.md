# Counter App

シンプルなカウンターアプリです。

🔗 **デモ**: [https://counter-mu-eight-35.vercel.app/](https://counter-mu-eight-35.vercel.app/)

---

## 使用技術

- **React** — UIコンポーネントの構築
- **JavaScript (ES6+)** — ロジック実装
- **CSS** — スタイリング
- **Vercel** — デプロイ

---

## 機能

- カウントの足し算（+1）
- カウントの引き算（-1）
- カウントの掛け算 (x1)
- カウントの割り算 (÷1)


## デプロイ

[Vercel](https://vercel.com/) にデプロイしています。  
`main` ブランチへのプッシュで自動デプロイされます。
