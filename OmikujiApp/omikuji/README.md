# 🎋 おみくじアプリ

> 和風でシンプルなおみくじWebアプリです!

🔗 **デモ:** https://omikuji-henna.vercel.app

---

## ✨ 機能

- ボタンひとつでランダムに運勢を抽選
- 大吉・中吉・吉・小吉・凶の5種類（それぞれ確率が違います）

---

## 🛠 技術スタック

| 項目 | 内容 |
|------|------|
| フレームワーク | React 18 |
| ビルドツール | Vite |
| スタイリング | CSS Modules |
| フォント | Google Fonts |
| デプロイ | Vercel |

---

## 🚀 ローカルで動かす

```bash
git clone https://github.com/your-username/omikuji-app.git
cd omikuji-app
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開く。

---

## 📁 ディレクトリ構成

```
src/
├── App.jsx       # メインコンポーネント
├── App.css       # 和風スタイル
├── main.jsx      # エントリーポイント
└── index.css     # グローバルスタイル
```
